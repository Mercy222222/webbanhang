# Nền tảng Kỹ thuật Frontend (Frontend Foundations)

- **`ui-frontend`**: Tập trung vào cấu trúc HTML ngữ nghĩa, CSS thuần và JS client-side. Lưới linh hoạt (Flexbox/Grid), tránh dùng `px` cứng. Sử dụng Glassmorphism và các vi-tương tác mượt mà.
- **`frontend-nextjs`**: React và Next.js (App Router). Khuyến khích dùng SWR/React Query.
- **`web-artifacts-builder`**: Xây dựng artifact phức tạp bằng React và Tailwind CSS.
- **`frontend-dev`**: Các trang Hero, Showcase với hiệu ứng điện ảnh và Generative art.
- **`shadcn-ui`**: Sử dụng thư viện `shadcn/ui` chuẩn accessibility.
