# Trình diễn & Lập trình Video (Presentations & Video Generation)

- **`frontend-slides`**: Bài thuyết trình HTML (Presentations) giàu hiệu ứng hoạt ảnh.
- **`remotion`**: Lập trình tạo video trực tiếp bằng React.
- **`video-hyperframes`**: Khung hoạt ảnh điện ảnh tự động (HTML/CSS) tương thích Remotion.
