# Kỹ thuật Kỹ sư Giao diện (UI Engineering & Guidelines)

- **`frontend-ui-engineering`**: Đề cao **Composition** (lắp ghép component), tuân thủ WCAG 2.1 AA. Xử lý triệt để các trạng thái Empty/Error/Loading.
- **`web-design-guidelines`**: Quy chuẩn thiết kế từ đội ngũ Vercel.
- **`apple-hig`**: Nguyên tắc thiết kế giao diện người dùng của Apple.
- **`design-wireframe`**: Xây dựng khung giao diện (Wireframing). Chuẩn hóa tỷ lệ font chữ (Typography hierarchy), cấu trúc Bento Grids, và hệ thống khoảng trắng (Spacing guidelines).
